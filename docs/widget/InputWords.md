## API

### InputWords

| 属性        | 描述                            | 是否必传 | 类型                        | 默认值 |
| :---------- | :------------------------------ | :------- | :-------------------------- | :----- |
| maxLength   | 最大输入长度                    | -        | `number`                    | -      |
| showCount   | 是否显示计数器                  | -        | `boolean`                   | -      |
| onChange    | 输入值变化回调函数              | -        | `(words: string[]) => void` | -      |
| width       | 宽度                            | -        | `string\|number`            | `100%` |
| ...其他属性 | 其他属性与 antd Select 组件一致 | -        | -                           | -      |
