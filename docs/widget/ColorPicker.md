# ColorPicker

> 自 `antd@5.5.0` 版本开始提供该组件。

## 使用场景

需要在应用中选择颜色的地方使用。

## API

### ColorPicker

| 属性         | 描述                     | 是否必传 | 类型                      |
| :----------- | :----------------------- | :------- | :------------------------ |
| value        | 当前颜色值               | 否       | `string`                  |
| onChange     | 颜色值变化回调函数       | 否       | `(value: string) => void` |
| widgetRender | 自定义渲染内容           | -        | `ReactNode`               |
| ...其他属性  | 与 antd ColorPicker 一致 | -        | -                         |
