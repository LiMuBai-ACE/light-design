## API

### InputCounter

| 属性        | 描述                     | 是否必传 | 类型     | 默认值                                                                                                                              |
| :---------- | :----------------------- | :------- | :------- | :---------------------------------------------------------------------------------------------------------------------------------- |
| placeholder | 占位文本                 | -        | `string` | 请输入数字                                                                                                                          |
| max         | 最大值                   | -        | `number` | [Number.MAX_SAFE_INTEGER](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Number/MAX_SAFE_INTEGER) |
| min         | 最小值                   | -        | `number` | 1                                                                                                                                   |
| ...其他属性 | 与 antd InputNumber 一致 | -        | -        | -                                                                                                                                   |
