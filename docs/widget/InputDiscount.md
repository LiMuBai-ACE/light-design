## API

### InputDiscount

| 属性        | 描述                     | 是否必传 | 类型     | 默认值     |
| :---------- | :----------------------- | :------- | :------- | :--------- |
| placeholder | 占位文本                 | -        | `string` | 请输入折扣 |
| max         | 最大值                   | -        | `number` | 9.99       |
| min         | 最小值                   | -        | `number` | 0.01       |
| ...其他属性 | 与 antd InputNumber 一致 | -        | -        | -          |
