## API

### LightForm

| 属性           | 描述           | 是否必传 | 类型                    |
| -------------- | -------------- | -------- | ----------------------- |
| value          | 是否选择       | 否       | `boolean` \| `undefine` |
| defaultChecked | 树形结构数据   | 否       | `boolean` \| `undefine` |
| disabled       | 禁用组件       | 否       | `boolean` \| `undefine` |
| onChange       | 状态改变时触发 | 否       | `(value) => void`       |
