# LightFormBuilder 表单生成器

## 使用场景

为了使代码更简洁，易用，快速生成一个表单

## 代码演示

```tsx
/**
 * transform: true
 * defaultShowCode: false
 * background: 'rgb(204,204,204, .33)'
 * padding: 20px
 */
import { LightFormBuilder } from 'light-design';
import React from 'react';
function demo() {
  return <LightFormBuilder />;
}

export default demo;
```
